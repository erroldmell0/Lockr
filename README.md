# Lockr

Lockr is a full-stack web application designed to securely manage user information using a modern web stack. The application features a React-based frontend and a Node.js backend, with MongoDB used for persistent data storage and MongoDB Compass for database management and inspection.

## Tech Stack

### Frontend
- React
- Vite
- HTML, CSS, JavaScript

### Backend
- Node.js
- Express

### Database
- MongoDB
- MongoDB Compass
